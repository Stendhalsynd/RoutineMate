# S4-3 Spec - 성능/접근성/회귀 안정화

## 범위
- Query 캐시 우선 표시, 동기화 상태 배지, 낙관 반영 강화.
- 저장 버튼 여백 규칙 통일(`action-gap-lg`).
- 상태 메시지 `aria-live` 적용.

## Given / When / Then
1. Given 페이지 이동, When 데이터 재요청 중, Then 기존 캐시를 유지해 빈 상태 깜빡임을 줄인다.
2. Given 저장 버튼 클릭, When 응답 대기 중, Then 즉시 상태 변경이 보인다.
3. Given 저장 실패, When 롤백, Then 오류 메시지가 즉시 노출된다.

## 실패 테스트
- 회귀: Day/Week/Month 라벨 불일치가 없어야 한다.

## 완료 기준
- e2e에서 페이지 전환/저장 즉시성 회귀 없음.
