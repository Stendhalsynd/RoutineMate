# S6-4 Spec - 기록 화면 정보 위계 재정렬

## 범위
- 모바일 기록 화면의 첫 카드 상단에는 `기록 날짜`, `오늘 상태`, `체중/체지방 빠른 입력`이 먼저 보인다.
- 식단/운동 체크인 상세 섹션은 빠른 입력 아래로 이동한다.
- 웹 상단 헤더 카피는 줄이고, 상태/액션과 실제 데이터 영역 사이의 간격을 줄인다.

## Given / When / Then
1. Given 모바일 사용자가 기록 탭에 진입할 때,
   When 첫 화면을 본다면,
   Then 날짜 선택, 오늘 상태 요약, 체성분 빠른 입력이 식단/운동 상세보다 먼저 보인다.
2. Given 웹 사용자가 workspace를 열 때,
   When 헤더가 렌더링되면,
   Then 긴 설명 문장보다 상태와 주요 액션이 우선 보인다.

## 완료 기준
- 모바일 기록 탭 진입 후 빠른 입력까지 추가 스크롤 없이 접근 가능하다.
- 웹 헤더의 보조 설명은 축소되고 첫 데이터 카드가 더 빨리 노출된다.
