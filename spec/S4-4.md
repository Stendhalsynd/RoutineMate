# S4-4 Spec - 웹+APK 동시 릴리즈 파이프라인

## 범위
- 웹: `deploy:prod:verify` 루프 유지.
- 모바일: EAS 기반 APK 빌드 프로필 추가.
- 릴리즈: GitHub Release에 APK 첨부 자동화 스크립트 제공.

## Given / When / Then
1. Given 릴리즈 태그, When APK 빌드 완료, Then 산출물을 Release에 업로드한다.
2. Given 웹 배포 성공, When Playwright 검증, Then canonical URL 시나리오 통과.

## 실패 테스트
- EAS 미설정(env 누락) 시 명확한 에러 메시지 출력.

## 완료 기준
- 배포/검증/업로드 스크립트가 실행 가능 상태.
