# S6-5 원통형 체성분 Picker + Granularity 추세

## 목표
- 모바일 체중/체지방 picker를 실제 원통형 무한 wheel처럼 동작하게 바꾼다.
- picker는 `00.0` 형식 미리보기만 보여주고 자리 라벨은 제거한다.
- picker를 열면 현재값이 있으면 현재값, 없으면 최근값, 그것도 없으면 최소값을 기본 선택으로 맞춘다.
- picker 안팎의 `최근값`, `-0.1`, `+0.1`, `정밀 선택` 빠른 액션은 모두 제거한다.
- 대시보드 체성분 추세는 `Day / Week / Month` granularity에 따라 일별 마지막값 / 주별 평균 / 월별 평균으로 집계한다.

## 수용 기준
1. 모바일 picker는 끝에서 멈추지 않고 계속 스크롤 가능한 구조를 가진다.
2. 모바일 picker는 최근값을 기본 선택으로 맞출 수 있는 초기 wheel index 계산 유틸을 가진다.
3. 모바일 picker는 허용 범위를 벗어난 조합을 확정값으로 만들지 않는다.
4. 모바일 picker UI에는 빠른 액션 버튼과 자리 라벨이 남아 있지 않아야 한다.
5. 체성분 추세는 `DashboardSummary.bodyMetricTrend` 필드를 유지한다.
6. `Day`는 동일 날짜의 마지막 기록을 사용한다.
7. `Week`와 `Month`는 기록된 값만 평균낸다.
8. 체중과 체지방은 각각 독립 축으로 평균된다.
