# S5-3 Spec - 체중/체지방 전체 구간 라인차트(웹+모바일)

## 범위
- `DashboardSummary`에 전체 체성분 시계열 `bodyMetricTrend`를 추가한다.
- 추세 데이터는 "최초 기록 ~ 최신 기록" 전체 구간을 날짜 오름차순으로 제공한다.
- 동일 날짜 중복 기록은 마지막 기록을 대표값으로 사용한다.
- 대시보드에 라인차트 2개(체중, 체지방)를 웹/모바일 모두 노출한다.

## Given / When / Then
1. Given 체성분 기록이 누적된 사용자, When 대시보드를 열면, Then 체중/체지방 라인차트가 전체 구간 추세를 표시한다.
2. Given range 토글(7d/30d/90d)을 변경, When 차트를 본다면, Then 체성분 라인차트 범위는 전체 구간으로 고정되어 유지된다.
3. Given 체중 또는 체지방 값이 비어 있는 날짜가 존재, When 차트 데이터를 생성, Then 없는 축 값은 `null`로 안전하게 처리된다.

## 실패 테스트
- body metrics가 0건이면 차트는 빈 상태 안내를 노출한다.
- 계약 테스트에서 `bodyMetricTrend` 타입이 누락되면 실패한다.

## 완료 기준
- API/도메인 타입(`DashboardSummary`)과 웹/모바일 렌더링이 동일 필드로 동작한다.
- 추세 데이터가 최초~최신 전체 기록 기준으로 일관되게 노출된다.
