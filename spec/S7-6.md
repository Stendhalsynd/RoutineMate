# S7-6 Body Metric Picker Visibility Fix

## Goal
- Fix the mobile decimal picker so selected weight/body-fat digits remain visible inside the highlighted band.

## Scope
- Move the highlight layer behind the wheel digits.
- Ensure the highlight overlay does not capture touches.
- Add a regression test that locks the source structure used to keep the selected digits visible.

## Acceptance Criteria
1. The selected number remains readable in the highlighted row.
2. The highlight layer is non-interactive.
3. Mobile unit tests include a regression assertion for the layer ordering.

## Verification
- `npm run test --workspace @routinemate/mobile`
- `npm run typecheck --workspace @routinemate/mobile`
